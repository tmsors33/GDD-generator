서비스 소개

구글 드라이브 문서 자동 생성기는 사용자가 입력한 문서 방향성에 따라 자동으로 구글 드라이브에 구글 독스 문서를 생성하는 애플리케이션입니다. 인공지능 기술을 활용하여 사용자가 원하는 방향성을 분석하고, 학습된 문서 데이터를 참조하여 더 관련성 높은 문서를 생성합니다.

주요 기능

  • 문서 자동 생성: 사용자 입력을 바탕으로 구글 드라이브에 문서 자동 생성
  • 문서 학습: 기존 문서를 시스템에 학습시켜 템플릿 생성 품질 향상
  • 인공지능 분석: OpenAI API를 활용한 텍스트 분석 및 구조화
  • 안전한 접근: 구글 OAuth 인증을 통한 안전한 드라이브 접근

문서 학습 시스템

문서 학습 시스템은 사용자가 업로드한 문서를 분석하여 벡터 데이터베이스에 저장합니다. 이렇게 학습된 데이터는 새로운 문서 생성 시 참조되어, 사용자의 기존 문서 스타일과 내용을 반영한 더 관련성 높은 문서를 생성할 수 있도록 도와줍니다.

지원 파일 형식

  • PDF 파일 (.pdf)
  • Word 문서 (.docx, .doc)
  • 텍스트 파일 (.txt)
  • Excel 파일 (.xlsx, .xls)

사용 기술

  • 백엔드: FastAPI, Python
  • AI 서비스: OpenAI API, LangChain
  • 벡터 저장소: ChromaDB
  • 문서 처리: Google Drive API, Google Docs API
  • 인증: Google OAuth 2.0
